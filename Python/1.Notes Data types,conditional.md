# 📦 1. Variables

A **variable** is like a box that stores data.

```python
name = "haridas das"
phone_number = 9273989823
have_eaten = True
```

We can check the type using:

```python
print(type(name))
```

---

# 🏷 2. Naming Conventions

Python follows these naming styles:

| Style       | Example        | Recommended              |
| ----------- | -------------- | ------------------------ |
| Snake Case  | `phone_number` | ✅ Yes                    |
| Camel Case  | `phoneNumber`  | ❌ Not standard in Python |
| Pascal Case | `PhoneNumber`  | ❌ Used for classes       |

👉 Always use **snake_case** for variables.

# ➗ 4. Arithmetic Operators

```python
x = 10
y = 3

print(x + y)  # Addition
print(x - y)  # Subtraction
print(x * y)  # Multiplication
print(x / y)  # Division
print(x % y)  # Remainder
print(x // y) # Integer division
```

---

# ⚖ 5. Conditional Statements

Used for decision-making.

```python
karma_points = 10

if karma_points > 75:
    print("Person is Dharmic")
else:
    print("Person is Adharmic")
```

---

# 🔍 Comparison Operators

| Operator | Meaning               |
| -------- | --------------------- |
| `<`      | Less than             |
| `>`      | Greater than          |
| `==`     | Equal to              |
| `<=`     | Less than or equal    |
| `>=`     | Greater than or equal |
